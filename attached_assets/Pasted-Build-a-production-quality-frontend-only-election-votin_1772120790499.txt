Build a production-quality frontend-only election voting system prototype using React + Vite + TypeScript + Tailwind CSS.

IMPORTANT:
• No backend implementation
• All data stored in mock local state or localStorage
• Architecture must be Firebase-ready (auth + firestore hooks placeholders)
• Focus on security UI and verification simulation
• Clean folder architecture
• Mobile responsive
• Professional modern UI
• Route guards and state guards must exist

APP STRUCTURE:

1. Public Pages
   • Landing page (checks election active state)
   • Election inactive screen
   • Candidate listing page
   • Candidate detail modal
   • Email verification page (simulate Firebase email link)
   • Camera identity verification modal (use browser camera API)
   • Voting confirmation screen
   • Duplicate voting blocked screen
   • Success message screen

2. Admin Pages
   • Admin login page (email link simulation)
   • Admin authorization screen
   • Admin dashboard with analytics cards
   • Candidate management page (add/edit/delete UI)
   • Election control page (start/end toggle)
   • Live voting monitor page (realtime simulation)
   • Results page with charts
   • Security audit panel

3. SECURITY UI REQUIREMENTS

   • Admin route guard

   • Device fingerprint UI indicator
   • Suspicious activity alert component
   • Session timer
   • Camera verification popup
   • Email verification pending screen
   • Identity verified badge

   • Tamper detection warning banner
